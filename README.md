<h1 align="center">ReDark</h1>

<div align="center">

一个基于 React 驱动的 UI 组件库 模板文件。

## 📦 先安装依赖

```bash
yarn
```

## 启动项目

```bash
yarn start
```

## 打包项目

```bash
yarn build
```

## 🔨 使用 redark 的组件

```jsx
import { Button } from '@rd'

const App = () => (
  <>
    <Button type="primary">PRESS ME</Button>
  </>
)
```
